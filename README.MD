# 黑马旅游网项目
## 技术选型
### 1.web层
    1.Servlet:前端控制器
    2.html：视图
    3.Filter：过滤器
    4.BeanUtils:数据库封装
    5.jackson:json序列化工具
### 2.Service层
    1.javamail:java发送邮件工具
    2.Redis:nosql内存数据库
    3.jedis：java的redis客户端
### 3.Dao层
    1.Mysql:数据库
    2.druid：数据库连接池
    3.jdbcTemplate：jdbc的工具
